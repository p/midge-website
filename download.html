Requirements

Midge has been developed and tested with the following software (earlier versions may work):

  • postgres-7.4.2
  • psycopg-1.1.11
  • python-2.3.3
  • egenix-mx-base-2.0.4

Download

Version 0.1 is now available. See the download page on source forge.

CVS access and other source forge facilities are available on the project page at source forge.

Installation

For details, see the README in the tarball. In short:
  • Install code using Python's distutils.
  • Edit configuration file.
  • Setup database tables.
  • Run tests (optional).
  • Run daemon.

In other words:

sudo python setup.py install vi /etc/midge.conf sudo midge-admin setup midge-test midged &
